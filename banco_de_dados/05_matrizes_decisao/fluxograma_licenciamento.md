```
Proprietário inscreve imóvel no CAR (SIMCAR)
        │
        ▼
CAR validado? ──(NÃO)──→ Retificar cadastro
        │
      (SIM)
        ▼
RL e APP delimitadas e regulares?
        │
      (SIM)                    (NÃO)
        ▼                        ▼
Protocola pedido de ASV      Aderir ao PRA primeiro
        │
        ▼
Análise documental (SEMA)
        │
        ▼
Documentação completa? ──(NÃO)──→ Complementação
        │
      (SIM)
        ▼
Vistoria técnica (quando exigida)
        │
        ▼
Parecer técnico favorável? ──(NÃO)──→ Indeferimento
        │
      (SIM)

        │
        ▼

        │
        ▼
Monitoramento via satélite (SLAPR)
```
