# Checklists e Erros Técnicos Recorrentes

tags: `checklist` `erros` `indeferimento` `documentação` `boas práticas`

## Checklist — Pedido de ASV

- [ ] CAR ativo no SIMCAR
- [ ] RL averbada/registrada (percentual correto para o bioma)
- [ ] APP delimitada e sem sobreposição com área de supressão
- [ ] Mapa georreferenciado (SIRGAS 2000, escala adequada)
- [ ] Memorial descritivo com coordenadas dos vértices
- [ ] Inventário florestal da área de supressão
- [ ] ART do responsável técnico
- [ ] Comprovação de titularidade ou posse do imóvel
- [ ] Compatibilidade com ZEE-MT
- [ ] Sem sobreposição com UC, TI ou área embargada
- [ ] Taxa de licenciamento paga

## Checklist — PMFS

- [ ] Inventário florestal (censo ou amostragem conforme categoria)
- [ ] Mapeamento das UPAs e UTs
- [ ] Descrição de técnicas de EIR (Exploração de Impacto Reduzido)
- [ ] Planejamento de infraestrutura (pátios, estradas, ramais de arraste)
- [ ] Lista de espécies a explorar com DMC (Diâmetro Mínimo de Corte)
- [ ] Ciclo de corte justificado
- [ ] Plano de proteção (contra fogo, invasão, caça)
- [ ] ART do engenheiro florestal

## Erros Técnicos Recorrentes

| Erro | Consequência |
|------|--------------|
| CAR não validado ou desatualizado | Processo não é aceito |
| Shapefile com erro topológico (sobreposição, gaps) | Rejeição do mapa |
| Área de supressão sobrepõe APP | Indeferimento imediato |
| RL abaixo do percentual mínimo sem adesão ao PRA | Indeferimento |
| Falta de ART | Processo devolvido |
| Inventário florestal inconsistente com a fitofisionomia | Parecer contrário |
| Coordenadas em DATUM errado (não SIRGAS 2000) | Rejeição do mapa |
| Memorial descritivo diverge do shapefile | Complementação exigida |
| Ausência de outorga de água quando há captação | Condicionante não atendida |
| Espécies protegidas no inventário sem justificativa | Análise especial exigida |
